# Herramienta Interfaz Descripción Ventajas
1

Kali Linux

https://www.kali.org/
Kali Linux es una distribución de Linux de código abierto basada en Debian que ofrece una variedad de software gratuito, utilidades de seguridad cibernética y herramientas de prueba de penetración
  • Permite a los usuarios ejecutar archivos ejecutables de Linux directamente desde un sistema Windows 10
  • Desarrolla aplicaciones web dinámicas.
  • ¡ Es Gratis!
2

KeePass

https://keepass.info/
Kali Linux es una distribución de Linux de código abierto basada en Debian que ofrece una variedad de software gratuito, utilidades de seguridad cibernética y herramientas de prueba de penetración
  • PEl cifrado de la base de datos utiliza los algoritmos de cifrado más seguros disponibles: AES-256, ChaCha20 y Twofish.
  • Cifra la base de datos completa, lo que significa que los nombres de usuario, notas, etc
3

Marco de Metasploit

https://www.metasploit.com/
Metasploit Framework es una herramienta de validación de vulnerabilidades y explotación que puede utilizar de forma ofensiva para probar sus sistemas en busca de vulnerabilidades conocidas y abiertas
  • Esta herramienta de seguridad de código abierto le ayuda a dividir el flujo
  • También es una herramienta de auditoría y escaneo de puertos de red
  • Se integra con librerías JavaScript.
  • Facilita enormemente el diseño de interfaces
4

Nikto

https://cirt.net/nikto2
Nikto es un escáner de servidor web gratuito y de código abierto, que escanea los servidores web en busca de múltiples vulnerabilidades.
  • Soporte completo de proxy HTTP.
  • Comprobaciones de componentes de servidor obsoletos
  • Motor de plantillas para personalizar fácilmente los informes
5

OpenVAS

https://www.openvas.org/
OpenVAS es un escáner de vulnerabilidades de código abierto y completo, de uso gratuito. Los usuarios pueden realizar pruebas no autenticadas y autenticadas para varios protocolos industriales y de Internet de alto y bajo nivel
  • Permite ajustar el rendimiento para escaneos a gran escala
  • Los usuarios pueden realizar cualquier tipo de vulnerabilidad
6

OSSEC

https://www.ossec.net/
OSSEC es un sistema de detección de intrusiones (HIDS) de código abierto, escalable y multiplataforma, cuyos creadores quieren mantenerse libres en el futuro previsible
  • Use OSSEC en las instalaciones y en la nube con el propósito de proteger el servidor o como una herramienta de análisis de registros
7

VeraCrypt

https://www.veracrypt.fr
VeraCrypt es una herramienta de seguridad para el cifrado de discos. Se ejecuta en Windows, Mac OSX y Linux y crea un disco virtual encriptado dentro de un archivo antes de montarlo como un disco real
  • Crea un disco virtual encriptado dentro de un archivo y lo monta como un disco real.
  • Cifra una partición completa o un dispositivo de almacenamiento , como una unidad flash USB o un disco duro.
8

Wireshark

https://www.wireshark.org/
Wireshark es una herramienta gratuita y de código abierto para el análisis de protocolos de red. Esta herramienta de ciberseguridad permite a los profesionales de seguridad observar la red a un nivel microscópico al ver el tráfico, descargar paquetes específicos, verificar el formato del paquete y encontrar problemas de red de esta manera.
  • Inspección profunda de cientos de protocolos, y se agregan más todo el tiempo
  • Captura en vivo y análisis fuera de línea
  • Navegador de paquetes estándar de tres paneles
