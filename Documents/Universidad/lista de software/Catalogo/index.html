Software libre

El software libre es una cuestión de la libertad de los usuarios de ejecutar, copiar, distribuir, estudiar, cambiar y mejorar el software.


Software privado

Cualquier programa informático en el que el código fuente no está disponible impidiendo que los usuarios puedan ejecutarlo, estudiarlo, modificarlo y copiarlo.
