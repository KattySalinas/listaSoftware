Atom

https://atom.io/
Es el editor de texto desarrollado por Github. Estéticamente llama mucho la atención porque cuidan mucho los detalles, además de ser muy fácil de usar. Muy similar al Sublime Text, pero con la diferencia de que Atom es gratuito.
  • Viene con un gestor de paquetes integrado, a través del cual puedes instalar o modificar plugins con total comodidad (o incluso crear los tuyos).
  • Atom te ayuda a escribir el código de forma rápida y flexible a través de su inteligente modo de autocompletado.

bootstrap

https://getbootstrap.com/
Originariamente utilizada por los desarrolladores de Twitter, Bootstrap fue toda una revolución, sobre todo para aquellos que no se atrevían a maquetar páginas web. Con este framework,
  • Incluye Grid system: muy útil para maquetar por columnas.
  • Sus plantillas son de sencilla adaptación responsive.
  • Se integra con librerías JavaScript.
  • Facilita enormemente el diseño de interfaces

jQuery

https://jquery.com/
Este framework de Javascript es el más utilizado en el mundo. La característica principal de esta biblioteca es que permite cambiar el contenido de una página web sin necesidad de recargarla, mediante la manipulación del árbol DOM y peticiones AJAX.
  • Query utiliza sintaxis muy parecida a CSS.
  • Funciona con series de elementos.
  • Permite manipular series de elementos y modificarlas con una simple línea de código.
  • Te ayuda a concentrarte en el resultado final.

Javascript

https://www.javascript.com/
Es el lenguaje de programación del lado del cliente por antonomasia. Es un lenguaje orientado a objetos, débilmente tipado y dinámico. Desde junio de este año está disponible la versión ECMAScript 6.
  • Es un lenguaje muy sencillo.
  • Es rápido, por lo tanto tiende a ejecutar las funciones inmediatamente.
  • Cuenta con múltiples opciones de efectos visuales.

BackboneJS

https://backbonejs.org/
BackboneJS es una librería para el desarrollo de sitios web, principalmente usada para aplicaciones web con bastante interacción con el cliente, donde se hace un uso intensivo de Javascript, Ajax, etc
  • Permite la programación atendiendo al paradigma MVC
  • Te ayuda a crear estructuras bien definidas para los datos de tu aplicación
  • Simplifica el uso de vistas, que te ayudan a renderizar interfaces
  • Permite el uso de colecciones de modelos

  • El uso de HTML como lenguaje declarativo hace que sea muy intuitivo.
